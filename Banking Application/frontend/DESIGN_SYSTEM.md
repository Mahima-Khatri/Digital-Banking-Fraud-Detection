# 🎨 Design System - Dark Theme

## Color Palette

### Background Colors
- `--bg-primary`: #0f1419 (Main background)
- `--bg-secondary`: #1a1f2e (Secondary background)
- `--bg-tertiary`: #252d3d (Tertiary background)
- `--bg-card`: #1e2738 (Card background)
- `--bg-hover`: #2a3447 (Hover state)

### Brand Colors
- `--primary`: #4c6ef5 (Primary blue)
- `--primary-dark`: #3b5bdb (Darker blue)
- `--primary-light`: #5c7cfa (Lighter blue)

### Status Colors
- `--success`: #51cf66 (Green - Safe transactions)
- `--warning`: #ffd43b (Yellow - Medium risk)
- `--danger`: #ff6b6b (Red - High risk/Fraud)
- `--info`: #4dabf7 (Blue - Information)

### Text Colors
- `--text-primary`: #e9ecef (Primary text)
- `--text-secondary`: #adb5bd (Secondary text)
- `--text-tertiary`: #868e96 (Tertiary text)

## Components

### Cards
- Background: var(--bg-card)
- Border: 1px solid var(--border-color)
- Border Radius: var(--radius-lg)
- Shadow: var(--shadow-md)

### Buttons
- Primary: var(--primary) background
- Hover: Lift effect + glow shadow
- Disabled: 60% opacity

### Inputs
- Background: var(--bg-tertiary)
- Border: var(--border-color)
- Focus: var(--primary) border + glow

## Animations
- Fade in: 0.3s ease
- Slide in: 0.3s cubic-bezier
- Hover lift: translateY(-2px)
- Glow effect: box-shadow with primary color

## Typography
- Headings: 600 weight
- Body: 400 weight
- Small text: 0.875rem
- Icons: 1.25rem

## Spacing
- Card padding: 1.5rem - 2rem
- Section gap: 1.5rem - 2rem
- Element gap: 1rem
